HOW TO USE THE SOFTWARE:

You can access a published web build here: https://tippacanoe.itch.io/blackjack-trainer
Alternatively, you can download [Blackjack-Trainer/Blackjack Trainer Build](https://github.com/tippacanoe/Blackjack-Trainer/tree/main/Blackjack%20Trainer%20Build). Then, execute Blackjack-Trainer-Build/index.html locally on your system. 

All files that the Unity editor utilizes to build the project are located in [Blackjack-Trainer/Blackjack Trainer](https://github.com/tippacanoe/Blackjack-Trainer/tree/main/Blackjack%20Trainer)
All source code for the project is located in [Blackjack-Trainer/Blackjack Trainer/Assets/Scripts](https://github.com/tippacanoe/Blackjack-Trainer/tree/main/Blackjack%20Trainer/Assets/Scripts)
